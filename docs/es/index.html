Documentación de led.run

Transforma cualquier pantalla en un letrero digital profesional. Sin aplicaciones, sin cuentas — solo una URL.

Cómo funciona

led.run convierte URLs en letreros digitales. La ruta es tu texto, la cadena de consulta es tu configuración. Abre la URL en cualquier navegador y tu letrero estará listo.

led.run/HELLO?theme=neon&color=ff00ff
Texto (ruta URL) Configuración (parámetros de consulta)

El flujo es simple: la URL se analiza en texto y parámetros, un renderizador de tema toma el control de la pantalla y tu letrero se muestra. Todo sucede del lado del cliente — sin procesamiento en servidor.

Protocolo URL

https://led.run/[text]?[params]
  • Texto — la ruta URL después de /. Los espacios pueden codificarse como + o %20.
  • Parámetros — cadena de consulta estándar. Todos los parámetros son "indicaciones de preferencia" opcionales — los temas deciden si los utilizan.

Ejemplos

led.run/HELLO                          Letrero simple
led.run/DO+NOT+DISTURB?t=do-not-disturb   Selección de tema
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   Colores personalizados
led.run/WELCOME?mode=flow&speed=120      Marquesina con desplazamiento

Modos de visualización

Los temas admiten dos modos de visualización:

Modo letrero

Visualización estática a pantalla completa. El texto se ajusta automáticamente para llenar el espacio disponible. Ideal para mensajes cortos como "OPEN", "ON AIR" o "DO NOT DISTURB".

Modo flujo

Marquesina con desplazamiento. El texto se desplaza continuamente por la pantalla. Ideal para mensajes más largos, anuncios o tickers.

Detección automática: Si no se establece el parámetro mode, la mayoría de los temas usan por defecto el modo letrero para texto corto (≤10 caracteres) y el modo flujo para texto más largo.

Parámetros globales

Estos parámetros son reconocidos por todos los temas. Cada tema proporciona sus propios valores predeterminados.

Parámetro Alias Tipo Predeterminado Descripción
theme t string default ID del tema (ver Temas)
mode — string auto sign (estático) o flow (desplazamiento)
color c hex según tema Color del texto (formato de color)
bg — hex según tema Color de fondo
speed — number según tema Velocidad de desplazamiento en px/s (modo flujo)
direction dir string left Dirección de desplazamiento: left o right
font — string según tema Nombre de la familia tipográfica
scale — number 1 Multiplicador de escala (0.1–1). Ver Escala
fill — hex según tema Fondo de la cara de tarjeta (temas tipo tarjeta, cuando scale < 1)
wakelock w boolean true Mantener pantalla encendida (evita suspensión)
cursor cur number — Retardo de ocultación automática del cursor en milisegundos
lang l string auto Idioma de la interfaz: en, zh, ja, ko, es, fr, de

Formato de color

Los colores se especifican como valores hexadecimales sin el prefijo #:

  • 6 dígitos RRGGBB — ej. ff0000 para rojo
  • 8 dígitos AARRGGBB — ej. 80ff0000 para rojo con 50% de transparencia
led.run/HELLO?color=00ff41          Texto verde
led.run/HELLO?bg=1a0033&c=ff00ff   Fondo púrpura, texto magenta

Escala y temas tipo tarjeta

El parámetro scale (0.1–1) controla el tamaño de visualización. Su comportamiento depende del tipo de tema:

Temas tipo tarjeta

Los temas tipo tarjeta (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) usan CSS transform: scale() en un contenedor interno. Cuando scale < 1, la tarjeta flota centrada sobre un fondo transparente. Usa el parámetro fill para establecer el color de la cara de la tarjeta, y bg para el fondo de la página.

Temas extendidos

Todos los demás temas aplican scale como multiplicador del tamaño de fuente: el resultado del ajuste automático se multiplica por el valor de escala. Los efectos de fondo permanecen a pantalla completa.

Interacciones

Una vez que se muestra un letrero, puedes interactuar con él usando teclado, ratón o pantalla táctil.

Teclado

  • Space Pausar / reanudar desplazamiento (modo flujo)
  • F Alternar pantalla completa
  • Esc Salir de pantalla completa

Ratón / Táctil

  • Clic / toque simple Pausar / reanudar
  • Doble clic / toque Alternar pantalla completa

Barra de herramientas

Una barra de herramientas flotante aparece en la parte inferior de la pantalla (se oculta automáticamente con el cursor). Proporciona:

  • Pantalla completa — entrar o salir del modo de pantalla completa
  • Rotar — rotar la pantalla 90° (útil para letreros verticales en pantallas horizontales)
  • Transmitir — transmitir a una pantalla externa vía Presentation API
  • Compartir — copiar la URL actual para compartir

Transmisión

La función de transmisión utiliza la W3C Presentation API para mostrar tu letrero en una pantalla secundaria. Actualmente compatible con Chrome y Edge.

  • Haz clic en el botón de transmisión en la barra de herramientas para seleccionar una pantalla
  • El receptor muestra una pantalla limpia sin barra de herramientas ni controles
  • La conexión persiste entre navegaciones de página
El botón de transmisión solo aparece en navegadores que admiten la Presentation API.
