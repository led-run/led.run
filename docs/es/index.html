Documentación de led.run

Convierte cualquier pantalla en un letrero de texto, efecto de luz, visualizador de sonido o pantalla de reloj. Sin aplicaciones, sin cuentas — solo una URL.

Cómo funciona

led.run utiliza URLs para crear pantallas. La ruta y la cadena de consulta configuran todo. Abre la URL en cualquier navegador y tu pantalla estará lista — sin procesamiento en servidor, sin necesidad de cuentas.

Letreros de texto

led.run/HELLO?theme=neon&color=ff00ff
Texto (ruta URL) Configuración (parámetros de consulta)

Efectos de luz

led.run/light?t=disco&speed=3
Prefijo de producto Efecto y configuración

Visualizadores de sonido

led.run/sound?t=bars&sensitivity=5
Prefijo de producto Visualizador y configuración

Pantallas de Reloj

led.run/time?t=digital&format=12h
Prefijo de producto Reloj y ajustes

Protocolo URL

Cuatro productos, cuatro patrones de URL:

Texto:  led.run/HELLO?t=neon              Ruta = contenido de texto
        led.run/text/HELLO?t=neon         Prefijo /text/ explícito
Luz:    led.run/light?t=disco             Prefijo de producto + parámetros
Sonido: led.run/sound?t=bars              Prefijo de producto + parámetros
Reloj:  led.run/time?t=digital            Prefijo de producto + parámetros
  • Texto — la ruta URL después de / se convierte en el texto mostrado. Los espacios pueden ser + o %20.
  • Luz, Sonido y Reloj — usan prefijos de ruta reservados. Toda la configuración se realiza mediante parámetros de consulta.
  • Todos los parámetros son “sugerencias de preferencia” opcionales — los temas/efectos/visualizadores deciden si los utilizan.
Prefijos reservados: text, light, sound, time están reservados. Para mostrar la palabra “light” como texto, usa led.run/text/light.

Texto — Descripción general

El producto Texto convierte cualquier ruta URL en una pantalla completa. Incluye 20 temas integrados, cada uno diseñado para un estilo y caso de uso específico. Los temas admiten dos modos de visualización: letrero estático y flujo con desplazamiento.

led.run/HELLO                          Letrero simple
led.run/DO+NOT+DISTURB?t=do-not-disturb   Selección de tema
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   Colores personalizados
led.run/WELCOME?mode=flow&speed=120      Marquesina con desplazamiento

Modos de visualización

Los temas de texto admiten dos modos de visualización:

Modo letrero

Pantalla estática a pantalla completa. El texto se ajusta automáticamente para llenar el espacio disponible. Ideal para mensajes cortos como “OPEN”, “ON AIR” o “DO NOT DISTURB”.

Modo flujo

Marquesina con desplazamiento. El texto se desplaza continuamente por la pantalla. Ideal para mensajes largos, anuncios o tickers.

Detección automática: Si no se establece el parámetro mode, la mayoría de los temas usan por defecto el modo sign para texto corto (≤10 caracteres) y el modo flow para texto más largo.

Parámetros de texto

Estos parámetros son reconocidos por los temas de texto. Cada tema proporciona sus propios valores por defecto.

Parámetro Alias Tipo Por defecto Descripción
theme t string default ID del tema (ver Temas)
mode — string auto sign (estático) o flow (desplazamiento)
color c hex según tema Color del texto (formato de color)
bg — hex según tema Color de fondo
speed — number según tema Velocidad de desplazamiento en px/s (modo flow)
direction dir string left Dirección de desplazamiento: left o right
font — string según tema Nombre de la familia tipográfica
scale — number 1 Multiplicador de escala (0.1–1). Ver Escala
fill — hex según tema Fondo de la tarjeta (temas de tarjeta, cuando scale < 1)
wakelock w boolean true Mantener pantalla encendida (evita suspensión)
cursor cur number — Retardo de ocultación automática del cursor en milisegundos
lang l string auto Idioma de la interfaz: en, zh, ja, ko, es, fr, de

Luz — Descripción general

El producto Luz convierte cualquier pantalla en un efecto de luz a pantalla completa. Úsalo como linterna, estroboscópico, luz ambiental o pantalla decorativa. 18 efectos integrados van desde lo práctico (sólido, SOS) hasta lo atmosférico (ondas de aurora, lámpara de lava).

led.run/light?t=solid              Linterna blanca
led.run/light?t=disco&speed=5      Luces disco rápidas
led.run/light?t=candle&warmth=9    Vela extra cálida
led.run/light?t=aurora-waves       Bandas de luz aurora
Nota: /light sin ?t= abre la página de inicio. Los enlaces para compartir siempre incluyen ?t=.

Sonido — Descripción general

El producto Sonido crea visualizaciones de audio en tiempo real controladas por tu micrófono. 12 visualizadores integrados transforman el sonido en efectos visuales fascinantes, desde barras de espectro clásicas hasta campos bokeh cinematográficos.

led.run/sound?t=bars              Analizador de espectro
led.run/sound?t=ocean&waveCount=10 Océano con más olas
led.run/sound?t=spikes            Picos radiales
Permiso de micrófono: Los visualizadores de sonido requieren acceso al micrófono. Tu navegador solicitará permiso en el primer uso. El audio se procesa localmente — no se graba ni se transmite.

Reloj — Resumen

El producto Reloj convierte tu pantalla en una pantalla de reloj a pantalla completa. Elige entre 14 temas de reloj integrados, desde LED clásico de 7 segmentos hasta tubos nixie y relojes solares. Cada tema muestra la hora actual con su propio estilo visual único.

led.run/time?t=digital                 Reloj digital predeterminado
led.run/time?t=nixie&color=ff8833      Reloj de tubos nixie
led.run/time?t=analog&style=modern     Reloj analógico moderno
led.run/time?t=lcd&format=12h          Reloj LCD, formato 12 horas
Nota: /time sin ?t= abre la página de inicio. Los enlaces para compartir siempre incluyen ?t=, ej. led.run/time?t=digital.

Color Format

Colors are specified as hex values without the # prefix:

  • 6-digit RRGGBB — e.g. ff0000 for red
  • 8-digit AARRGGBB — e.g. 80ff0000 for 50% transparent red
led.run/HELLO?color=00ff41          Green text
led.run/HELLO?bg=1a0033&c=ff00ff   Purple bg, magenta text

Scale & Card Themes

The scale parameter (0.1–1) controls display size. Its behavior depends on the theme type:

Card Themes

Card themes (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) use CSS transform: scale() on an inner wrapper. When scale < 1, the card floats centered on a transparent background. Use the fill param to set the card face color, and bg for the page background.

Extended Themes

All other themes apply scale as a font-size multiplier: the auto-fit result is multiplied by the scale value. Background effects remain fullscreen.

Time Product

Clock themes support scale, position, and padding as layout parameters. When scale < 1 or padding > 0, a wrapper is created inside the container with real dimensions (not CSS transform), so themes render at the correct size. Use position to place the clock (e.g., top-left), and padding to add spacing from container edges.

led.run/time?t=digital&scale=0.5&position=top-left&padding=3

Casting

The Cast feature uses the W3C Presentation API to display your content on a secondary screen. Currently supported in Chrome and Edge.

  • Click the Cast button in the toolbar to select a display
  • The receiver shows a clean display without toolbar or controls
  • The connection persists across page navigations
The Cast button only appears in browsers that support the Presentation API.
