led.run Documentation

Transform any screen into a professional digital sign. No apps, no accounts — just a URL.

How It Works

led.run turns URLs into digital signs. The path is your text, the query string is your configuration. Open the URL in any browser and your sign is ready.

led.run/HELLO?theme=neon&color=ff00ff
Text (URL path) Configuration (query params)

The flow is simple: the URL is parsed into text and parameters, a theme renderer takes over the screen, and your sign is displayed. Everything happens client-side — no server processing.

URL Protocol

https://led.run/[text]?[params]
  • Text — the URL path after /. Spaces can be encoded as + or %20.
  • Params — standard query string. All parameters are optional "preference hints" — themes decide whether to use them.

Examples

led.run/HELLO                          Simple sign
led.run/DO+NOT+DISTURB?t=do-not-disturb   Theme selection
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   Custom colors
led.run/WELCOME?mode=flow&speed=120      Scrolling marquee

Display Modes

Themes support two display modes:

Sign Mode

Static full-screen display. Text is auto-fitted to fill the available space. Ideal for short messages like "OPEN", "ON AIR", or "DO NOT DISTURB".

Flow Mode

Scrolling marquee. Text continuously scrolls across the screen. Ideal for longer messages, announcements, or tickers.

Auto-detection: If no mode param is set, most themes default to sign mode for short text (≤10 characters) and flow mode for longer text.

Global Parameters

These parameters are recognized by all themes. Each theme provides its own defaults.

Param Alias Type Default Description
theme t string default Theme ID (see Themes)
mode — string auto sign (static) or flow (scrolling)
color c hex per theme Text color (color format)
bg — hex per theme Background color
speed — number per theme Scroll speed in px/s (flow mode)
direction dir string left Scroll direction: left or right
font — string per theme Font family name
scale — number 1 Display scale multiplier (0.1–1). See Scale
fill — hex per theme Card face background (card themes, when scale < 1)
wakelock w boolean true Keep screen on (prevents sleep)
cursor cur number — Cursor auto-hide delay in milliseconds

Color Format

Colors are specified as hex values without the # prefix:

  • 6-digit RRGGBB — e.g. ff0000 for red
  • 8-digit AARRGGBB — e.g. 80ff0000 for 50% transparent red
led.run/HELLO?color=00ff41          Green text
led.run/HELLO?bg=1a0033&c=ff00ff   Purple bg, magenta text

Scale & Card Themes

The scale parameter (0.1–1) controls display size. Its behavior depends on the theme type:

Card Themes

Card themes (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) use CSS transform: scale() on an inner wrapper. When scale < 1, the card floats centered on a transparent background. Use the fill param to set the card face color, and bg for the page background.

Extended Themes

All other themes apply scale as a font-size multiplier: the auto-fit result is multiplied by the scale value. Background effects remain fullscreen.

Interactions

Once a sign is displayed, you can interact with it using keyboard, mouse, or touch.

Keyboard

  • Space Pause / resume scrolling (flow mode)
  • F Toggle fullscreen
  • Esc Exit fullscreen

Mouse / Touch

  • Single click / tap Pause / resume
  • Double click / tap Toggle fullscreen

Toolbar

A floating toolbar appears at the bottom of the screen (auto-hides with the cursor). It provides:

  • Fullscreen — enter or exit fullscreen mode
  • Rotate — rotate the display 90° (useful for portrait signs on landscape screens)
  • Cast — cast to an external display via Presentation API
  • Share — copy the current URL for sharing

Casting

The Cast feature uses the W3C Presentation API to display your sign on a secondary screen. Currently supported in Chrome and Edge.

  • Click the Cast button in the toolbar to select a display
  • The receiver shows a clean display without toolbar or controls
  • The connection persists across page navigations
The Cast button only appears in browsers that support the Presentation API.
