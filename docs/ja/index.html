led.run ドキュメント

あらゆる画面をプロフェッショナルなデジタルサインに変換。アプリもアカウントも不要 — URL だけで完結します。

仕組み

led.run は URL をデジタルサインに変換します。パスがテキスト、クエリ文字列が設定です。ブラウザで URL を開くだけで、サインが表示されます。

led.run/HELLO?theme=neon&color=ff00ff
テキスト（URL パス） 設定（クエリパラメータ）

フローはシンプルです：URL がテキストとパラメータに解析され、テーマレンダラーが画面を引き継ぎ、サインが表示されます。すべてクライアントサイドで処理され、サーバー処理は不要です。

URL プロトコル

https://led.run/[text]?[params]
  • Text — / 以降の URL パスです。スペースは + または %20 でエンコードできます。
  • Params — 標準的なクエリ文字列です。すべてのパラメータはオプションの「プリファレンスヒント」であり、テーマがそれらを使用するかどうかを決定します。

例

led.run/HELLO                          シンプルなサイン
led.run/DO+NOT+DISTURB?t=do-not-disturb   テーマの選択
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   カスタムカラー
led.run/WELCOME?mode=flow&speed=120      スクロールマーキー

表示モード

テーマは2つの表示モードに対応しています：

Sign モード

静的なフルスクリーン表示。テキストは利用可能なスペースに合わせて自動フィットされます。「OPEN」「ON AIR」「DO NOT DISTURB」などの短いメッセージに最適です。

Flow モード

スクロールマーキー。テキストが画面を横切って連続スクロールします。長いメッセージ、アナウンス、ティッカーに最適です。

自動検出： mode パラメータが設定されていない場合、ほとんどのテーマは短いテキスト（10文字以下）では sign モード、長いテキストでは flow モードをデフォルトとします。

グローバルパラメータ

これらのパラメータはすべてのテーマで認識されます。各テーマは独自のデフォルト値を提供します。

パラメータ エイリアス 型 デフォルト 説明
theme t string default テーマ ID（テーマ一覧を参照）
mode — string auto sign（静的）または flow（スクロール）
color c hex テーマ依存 テキストカラー（カラー形式）
bg — hex テーマ依存 背景色
speed — number テーマ依存 スクロール速度（px/秒、flow モード）
direction dir string left スクロール方向：left または right
font — string テーマ依存 フォントファミリー名
scale — number 1 表示スケール倍率（0.1–1）。スケールを参照
fill — hex テーマ依存 カード表面の背景色（カードテーマ、scale < 1 の場合）
wakelock w boolean true 画面を常にオンに保持（スリープ防止）
cursor cur number — カーソル自動非表示の遅延時間（ミリ秒）
lang l string auto UI 言語：en、zh、ja、ko、es、fr、de

カラー形式

カラーは # プレフィックスなしの16進数値で指定します：

  • 6桁 RRGGBB — 例：ff0000 で赤
  • 8桁 AARRGGBB — 例：80ff0000 で50%透過の赤
led.run/HELLO?color=00ff41          緑色のテキスト
led.run/HELLO?bg=1a0033&c=ff00ff   紫の背景、マゼンタのテキスト

スケール & カードテーマ

scale パラメータ（0.1–1）は表示サイズを制御します。動作はテーマの種類によって異なります：

カードテーマ

カードテーマ（broadcast、street-sign、wood、do-not-disturb、marquee、dot-matrix）は、内部ラッパーに CSS transform: scale() を使用します。scale < 1 の場合、カードは透過背景の中央にフロートします。fill パラメータでカード表面の色を、bg でページの背景色を設定します。

拡張テーマ

その他すべてのテーマは、scale をフォントサイズの倍率として適用します：自動フィットの結果にスケール値を乗算します。背景エフェクトはフルスクリーンのまま維持されます。

操作方法

サインが表示されたら、キーボード、マウス、またはタッチで操作できます。

キーボード

  • Space スクロールの一時停止 / 再開（flow モード）
  • F フルスクリーンの切り替え
  • Esc フルスクリーンの解除

マウス / タッチ

  • シングルクリック / タップ 一時停止 / 再開
  • ダブルクリック / タップ フルスクリーンの切り替え

ツールバー

画面下部にフローティングツールバーが表示されます（カーソルに連動して自動非表示）。以下の機能を提供します：

  • フルスクリーン — フルスクリーンモードの開始または解除
  • 回転 — ディスプレイを90度回転（横向き画面でのポートレートサインに便利）
  • キャスト — Presentation API 経由で外部ディスプレイにキャスト
  • 共有 — 現在の URL をコピーして共有

キャスト

キャスト機能は W3C Presentation API を使用して、サインをセカンダリスクリーンに表示します。現在、Chrome と Edge で対応しています。

  • ツールバーのキャストボタンをクリックしてディスプレイを選択
  • レシーバーはツールバーやコントロールなしのクリーンな表示
  • 接続はページ遷移後も維持されます
キャストボタンは Presentation API をサポートするブラウザでのみ表示されます。
