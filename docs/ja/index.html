led.run ドキュメント

あらゆる画面をテキストサイン、ライトエフェクト、サウンドビジュアライザー、またはクロックディスプレイに変換。アプリ不要、アカウント不要 — URLだけで完結。

仕組み

led.run はURLでディスプレイを作成します。パスとクエリ文字列ですべてを設定できます。ブラウザでURLを開くだけでディスプレイの準備完了 — サーバー処理もアカウントも不要です。

テキストサイン

led.run/HELLO?theme=neon&color=ff00ff
テキスト（URLパス） 設定（クエリパラメータ）

ライトエフェクト

led.run/light?t=disco&speed=3
プロダクトプレフィックス エフェクト & 設定

サウンドビジュアライザー

led.run/sound?t=bars&sensitivity=5
プロダクトプレフィックス ビジュアライザー & 設定

クロックディスプレイ

led.run/time?t=digital&format=12h
プロダクトプレフィックス 時計と設定

URLプロトコル

4つのプロダクト、4つのURLパターン：

Text:   led.run/HELLO?t=neon              パス = テキスト内容
        led.run/text/HELLO?t=neon         明示的な /text/ プレフィックス
Light:  led.run/light?t=disco             プロダクトプレフィックス + パラメータ
Sound:  led.run/sound?t=bars              プロダクトプレフィックス + パラメータ
Time:   led.run/time?t=digital            プロダクトプレフィックス + パラメータ
  • Text — / 以降のURLパスが表示テキストになります。スペースは + または %20 で指定できます。
  • Light、Sound & Time — 予約済みパスプレフィックスを使用します。すべての設定はクエリパラメータで行います。
  • すべてのパラメータは任意の「プリファレンスヒント」です — テーマ／エフェクト／ビジュアライザー／時計が使用するかどうかを決定します。
予約済みプレフィックス： text、light、sound、time は予約されています。「light」という文字をテキストとして表示するには、led.run/text/light を使用してください。

テキスト — 概要

テキストプロダクトは任意のURLパスをフルスクリーンディスプレイに変換します。20種の内蔵テーマを搭載し、それぞれ特定の雰囲気と用途に合わせて設計されています。テーマは2つの表示モードに対応：静止サインとスクロールフロー。

led.run/HELLO                          シンプルなサイン
led.run/DO+NOT+DISTURB?t=do-not-disturb   テーマ選択
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   カスタムカラー
led.run/WELCOME?mode=flow&speed=120      スクロールマーキー

表示モード

テキストテーマは2つの表示モードに対応しています：

サインモード

静止フルスクリーン表示。テキストは利用可能なスペースに自動フィットされます。「OPEN」「ON AIR」「DO NOT DISTURB」のような短いメッセージに最適です。

フローモード

スクロールマーキー。テキストが画面を連続的にスクロールします。長いメッセージ、お知らせ、ティッカーに最適です。

自動検出： mode パラメータが設定されていない場合、ほとんどのテーマは短いテキスト（10文字以下）では sign モード、長いテキストでは flow モードがデフォルトになります。

テキストパラメータ

これらのパラメータはテキストテーマで認識されます。各テーマは独自のデフォルト値を提供します。

パラメータ エイリアス 型 デフォルト 説明
theme t string default テーマID（テーマを参照）
mode — string auto sign（静止）または flow（スクロール）
color c hex テーマ毎 テキスト色（カラー形式）
bg — hex テーマ毎 背景色
speed — number テーマ毎 スクロール速度 px/s（フローモード）
direction dir string left スクロール方向：left または right
font — string テーマ毎 フォントファミリー名
scale — number 1 表示スケール倍率（Text: 0.1–1, Time: 0.1–3）。スケールを参照
fill — hex テーマ毎 カード面の背景色（カードテーマ、scale < 1 の場合）
wakelock w boolean true 画面をオンに保持（スリープ防止）
cursor cur number — カーソル自動非表示の遅延（ミリ秒）
lang l string auto UI言語：en、zh、ja、ko、es、fr、de

ライト — 概要

ライトプロダクトはあらゆる画面をフルスクリーンライトエフェクトに変換します。懐中電灯、ストロボ、ムードライト、アンビエントディスプレイとして使用できます。18種の内蔵エフェクトは実用的なもの（solid、SOS）から雰囲気のあるもの（aurora waves、lava lamp）まで揃っています。

led.run/light?t=solid              白色懐中電灯
led.run/light?t=disco&speed=5      高速ディスコライト
led.run/light?t=candle&warmth=9    暖かみ増しキャンドル
led.run/light?t=aurora-waves       オーロラライトバンド
注意： /light に ?t= がない場合はランディングページが開きます。共有リンクには常に ?t= が含まれます。

サウンド — 概要

サウンドプロダクトはマイクで駆動するリアルタイムオーディオビジュアライゼーションを作成します。12種の内蔵ビジュアライザーが音をクラシックなスペクトラムバーから映画的なボケフィールドまで、魅力的なビジュアルに変換します。

led.run/sound?t=bars              スペクトラムアナライザー
led.run/sound?t=ocean&waveCount=10 波を増やしたオーシャン
led.run/sound?t=spikes            放射状スパイク
マイクの許可： サウンドビジュアライザーはマイクへのアクセスが必要です。初回使用時にブラウザが許可を求めます。音声はローカルで処理され — 録音や送信は行われません。

時計 — 概要

Timeプロダクトは画面をフルスクリーン時計ディスプレイに変換します。クラシックな7セグメントLEDからニキシー管や日時計まで、14種の内蔵時計テーマから選べます。各テーマはそれぞれ独自のビジュアルスタイルで現在の時刻を表示します。

led.run/time?t=digital                 デフォルトデジタル時計
led.run/time?t=nixie&color=ff8833      ニキシー管時計
led.run/time?t=analog&style=modern     モダンアナログ時計
led.run/time?t=lcd&format=12h          LCD腕時計、12時間制
注意： /time に ?t= を付けないとランディングページにルーティングされます。共有リンクには常に ?t= を含めてください。例：led.run/time?t=digital。

Color Format

Colors are specified as hex values without the # prefix:

  • 6-digit RRGGBB — e.g. ff0000 for red
  • 8-digit AARRGGBB — e.g. 80ff0000 for 50% transparent red
led.run/HELLO?color=00ff41          Green text
led.run/HELLO?bg=1a0033&c=ff00ff   Purple bg, magenta text

Scale & Card Themes

The scale parameter controls display size. Its behavior depends on the theme type:

Card Themes

Card themes (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) use CSS transform: scale() on an inner wrapper. When scale < 1, the card floats centered on a transparent background. Use the fill param to set the card face color, and bg for the page background.

Extended Themes

All other themes apply scale as a font-size multiplier: the auto-fit result is multiplied by the scale value. Background effects remain fullscreen.

Time Product

Clock themes support scale (0.1–3), position, and padding as layout parameters. When scale ≠ 1 or padding > 0, a wrapper is created inside the container with real dimensions (not CSS transform), so themes render at the correct size. Values < 1 shrink the clock; values > 1 enlarge it beyond the container, with overflow:hidden clipping the excess. Use position to control which portion of the enlarged clock is visible (e.g., top-left anchors the top-left corner). padding adds spacing from container edges.

led.run/time?t=digital&scale=0.5&position=top-left&padding=3
led.run/time?t=digital&scale=2&position=center            — 2× zoom, centered
led.run/time?t=digital&scale=2&position=top-left           — 2× zoom, top-left anchor

The Cast feature uses the W3C Presentation API to display your content on a secondary screen. Currently supported in Chrome and Edge.

  • The connection persists across page navigations
