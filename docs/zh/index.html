led.run 文档

将任何屏幕变成文字标牌、灯光效果、声音可视化或时钟显示。无需应用，无需账号 — 只需一个 URL。

工作原理

led.run 使用 URL 创建显示。路径和查询字符串即可配置一切。在任何浏览器中打开 URL，显示即刻就绪 — 无需服务器处理，无需账号。

文字标牌

led.run/HELLO?theme=neon&color=ff00ff
文字（URL 路径） 配置（查询参数）

灯光效果

led.run/light?t=disco&speed=3
产品前缀 效果与设置

声音可视化

led.run/sound?t=bars&sensitivity=5
产品前缀 可视化与设置

时钟显示

led.run/time?t=digital&format=12h
产品前缀 时钟与设置

URL 协议

四种产品，四种 URL 格式：

Text:   led.run/HELLO?t=neon              路径 = 文字内容
        led.run/text/HELLO?t=neon         显式 /text/ 前缀
Light:  led.run/light?t=disco             产品前缀 + 参数
Sound:  led.run/sound?t=bars              产品前缀 + 参数
Time:   led.run/time?t=digital            产品前缀 + 参数
  • Text — / 之后的 URL 路径即为显示文字。空格可使用 + 或 %20。
  • Light、Sound 与 Time — 使用保留的路径前缀。所有配置通过查询参数完成。
  • 所有参数都是可选的"偏好提示" — 由主题/效果/可视化/时钟决定是否使用。
保留前缀：text、light、sound、time 为保留前缀。如需将"light"作为文字显示，请使用 led.run/text/light。

文字 — 概览

文字产品可将任何 URL 路径转为全屏显示。内置 20 种主题，每种都为特定风格和使用场景设计。主题支持两种显示模式：静态标牌和滚动走马灯。

led.run/HELLO                          简单标牌
led.run/DO+NOT+DISTURB?t=do-not-disturb   选择主题
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   自定义颜色
led.run/WELCOME?mode=flow&speed=120      滚动走马灯

显示模式

文字主题支持两种显示模式：

标牌模式（Sign）

静态全屏显示。文字自动适应可用空间。适用于短消息，如"OPEN"、"ON AIR"或"DO NOT DISTURB"。

滚动模式（Flow）

滚动走马灯。文字在屏幕上持续滚动。适用于较长的消息、公告或滚动字幕。

自动检测：如果未设置 mode 参数，大多数主题默认对短文本（≤10 个字符）使用标牌模式，对较长文本使用滚动模式。

文字参数

以下参数被文字主题识别。每个主题提供自己的默认值。

参数 别名 类型 默认值 说明
theme t string default 主题 ID（参见主题）
mode — string auto sign（静态）或 flow（滚动）
color c hex 随主题 文字颜色（颜色格式）
bg — hex 随主题 背景颜色
speed — number 随主题 滚动速度，单位 px/s（滚动模式）
direction dir string left 滚动方向：left 或 right
font — string 随主题 字体名称
scale — number 1 显示缩放倍率（0.1–1）。参见缩放
fill — hex 随主题 卡片正面背景色（卡片主题，scale < 1 时生效）
wakelock w boolean true 保持屏幕常亮（防止休眠）
cursor cur number — 光标自动隐藏延迟（毫秒）
lang l string auto 界面语言：en、zh、ja、ko、es、fr、de

灯光 — 概览

灯光产品可将任何屏幕变为全屏灯光效果。用作手电筒、频闪灯、氛围灯或装饰显示。18 种内置效果涵盖实用（纯色、SOS）到氛围（极光波浪、熔岩灯）。

led.run/light?t=solid              白色手电筒
led.run/light?t=disco&speed=5      快速迪斯科灯
led.run/light?t=candle&warmth=9    超温暖烛光
led.run/light?t=aurora-waves       极光光带
注意：/light 不带 ?t= 会打开首页。分享链接始终包含 ?t=。

时钟 — 概览

时钟产品将您的屏幕变为全屏时钟显示。可从 14 种内置时钟主题中选择，从经典七段 LED 到辉光管和日晷。每种主题以独特的视觉风格呈现当前时间。

led.run/time?t=digital                 默认数字时钟
led.run/time?t=nixie&color=ff8833      辉光管时钟
led.run/time?t=analog&style=modern     现代模拟时钟
led.run/time?t=lcd&format=12h          LCD 手表，12 小时制
注意：/time 不带 ?t= 参数将跳转到首页。分享链接始终包含 ?t=，例如 led.run/time?t=digital。

Color Format

Colors are specified as hex values without the # prefix:

  • 6-digit RRGGBB — e.g. ff0000 for red
  • 8-digit AARRGGBB — e.g. 80ff0000 for 50% transparent red
led.run/HELLO?color=00ff41          Green text
led.run/HELLO?bg=1a0033&c=ff00ff   Purple bg, magenta text

Scale & Card Themes

The scale parameter (0.1–1) controls display size. Its behavior depends on the theme type:

Card Themes

Card themes (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) use CSS transform: scale() on an inner wrapper. When scale < 1, the card floats centered on a transparent background. Use the fill param to set the card face color, and bg for the page background.

Extended Themes

All other themes apply scale as a font-size multiplier: the auto-fit result is multiplied by the scale value. Background effects remain fullscreen.

Time Product

Clock themes support scale, position, and padding as layout parameters. When scale < 1 or padding > 0, a wrapper is created inside the container with real dimensions (not CSS transform), so themes render at the correct size. Use position to place the clock (e.g., top-left), and padding to add spacing from container edges.

led.run/time?t=digital&scale=0.5&position=top-left&padding=3
