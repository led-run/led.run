led.run 文档

将任何屏幕变成专业的数字标牌。无需安装应用，无需注册账号 — 只需一个 URL。

工作原理

led.run 将 URL 转换为数字标牌。路径即是你的文字，查询参数即是你的配置。在任何浏览器中打开该 URL，你的标牌就准备好了。

led.run/HELLO?theme=neon&color=ff00ff
文字（URL 路径） 配置（查询参数）

流程非常简单：URL 被解析为文字和参数，主题渲染器接管屏幕，你的标牌随即显示。一切都在客户端完成 — 无需服务器处理。

URL 协议

https://led.run/[text]?[params]
  • Text — / 之后的 URL 路径。空格可编码为 + 或 %20。
  • Params — 标准查询字符串。所有参数都是可选的"偏好提示" — 由主题决定是否使用。

示例

led.run/HELLO                          简单标牌
led.run/DO+NOT+DISTURB?t=do-not-disturb   选择主题
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   自定义颜色
led.run/WELCOME?mode=flow&speed=120      滚动字幕

显示模式

主题支持两种显示模式：

静态模式（Sign Mode）

静态全屏显示。文字自动适配以填满可用空间。适合短消息，如"OPEN"、"ON AIR"或"DO NOT DISTURB"。

滚动模式（Flow Mode）

滚动字幕。文字在屏幕上持续滚动。适合较长的消息、公告或滚动信息。

自动检测：如果未设置 mode 参数，大多数主题默认对短文本（≤10 个字符）使用静态模式，对较长文本使用滚动模式。

全局参数

以下参数被所有主题识别。每个主题提供自己的默认值。

参数 别名 类型 默认值 说明
theme t string default 主题 ID（参见主题）
mode — string auto sign（静态）或 flow（滚动）
color c hex 因主题而异 文字颜色（颜色格式）
bg — hex 因主题而异 背景颜色
speed — number 因主题而异 滚动速度，单位 px/s（滚动模式）
direction dir string left 滚动方向：left 或 right
font — string 因主题而异 字体名称
scale — number 1 显示缩放倍数（0.1–1）。参见缩放
fill — hex 因主题而异 卡片正面背景色（卡片主题，当 scale < 1 时）
wakelock w boolean true 保持屏幕常亮（防止休眠）
cursor cur number — 光标自动隐藏延迟，单位毫秒
lang l string auto 界面语言：en、zh、ja、ko、es、fr、de

颜色格式

颜色以十六进制值指定，不带 # 前缀：

  • 6 位 RRGGBB — 例如 ff0000 表示红色
  • 8 位 AARRGGBB — 例如 80ff0000 表示 50% 透明红色
led.run/HELLO?color=00ff41          绿色文字
led.run/HELLO?bg=1a0033&c=ff00ff   紫色背景，品红色文字

缩放与卡片主题

scale 参数（0.1–1）控制显示大小。其行为取决于主题类型：

卡片主题

卡片主题（broadcast、street-sign、wood、do-not-disturb、marquee、dot-matrix）在内部包装元素上使用 CSS transform: scale()。当 scale < 1 时，卡片居中浮动在透明背景上。使用 fill 参数设置卡片正面颜色，使用 bg 设置页面背景。

扩展主题

所有其他主题将 scale 作为字体大小乘数：自动适配结果乘以缩放值。背景效果保持全屏。

交互操作

标牌显示后，你可以通过键盘、鼠标或触控进行交互。

键盘

  • Space 暂停 / 恢复滚动（滚动模式）
  • F 切换全屏
  • Esc 退出全屏

鼠标 / 触控

  • 单击 / 轻触 暂停 / 恢复
  • 双击 / 双击轻触 切换全屏

工具栏

屏幕底部会显示一个浮动工具栏（随光标自动隐藏）。它提供：

  • 全屏 — 进入或退出全屏模式
  • 旋转 — 旋转显示 90°（适用于在横屏上显示竖向标牌）
  • 投屏 — 通过 Presentation API 投屏到外部显示器
  • 分享 — 复制当前 URL 用于分享

投屏

投屏功能使用 W3C Presentation API 将你的标牌显示在第二块屏幕上。目前支持 Chrome 和 Edge 浏览器。

  • 点击工具栏中的投屏按钮选择显示器
  • 接收端显示干净的画面，不含工具栏和控件
  • 连接在页面跳转时保持不断
投屏按钮仅在支持 Presentation API 的浏览器中显示。
