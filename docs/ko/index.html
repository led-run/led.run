led.run 문서

어떤 화면이든 전문적인 디지털 사이니지로 변환하세요. 앱도, 계정도 필요 없습니다 — URL 하나면 됩니다.

작동 방식

led.run은 URL을 디지털 사이니지로 변환합니다. 경로가 텍스트이고, 쿼리 스트링이 설정입니다. 아무 브라우저에서 URL을 열면 사이니지가 준비됩니다.

led.run/HELLO?theme=neon&color=ff00ff
텍스트 (URL 경로) 설정 (쿼리 매개변수)

흐름은 간단합니다: URL이 텍스트와 매개변수로 파싱되고, 테마 렌더러가 화면을 차지하며, 사이니지가 표시됩니다. 모든 것이 클라이언트 측에서 처리됩니다 — 서버 처리가 필요 없습니다.

URL 프로토콜

https://led.run/[text]?[params]
  • 텍스트 — / 뒤의 URL 경로. 공백은 + 또는 %20으로 인코딩할 수 있습니다.
  • 매개변수 — 표준 쿼리 스트링. 모든 매개변수는 선택적 "선호 힌트"입니다 — 테마가 사용 여부를 결정합니다.

예시

led.run/HELLO                          간단한 사이니지
led.run/DO+NOT+DISTURB?t=do-not-disturb   테마 선택
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   커스텀 색상
led.run/WELCOME?mode=flow&speed=120      스크롤 마키

디스플레이 모드

테마는 두 가지 디스플레이 모드를 지원합니다:

사인 모드

정적 전체 화면 표시. 텍스트가 사용 가능한 공간에 맞게 자동 조절됩니다. "OPEN", "ON AIR", "DO NOT DISTURB" 같은 짧은 메시지에 적합합니다.

플로우 모드

스크롤 마키. 텍스트가 화면을 가로질러 연속적으로 스크롤됩니다. 긴 메시지, 공지사항 또는 티커에 적합합니다.

자동 감지: mode 매개변수가 설정되지 않은 경우, 대부분의 테마는 짧은 텍스트(≤10자)에 대해 sign 모드를, 긴 텍스트에 대해 flow 모드를 기본으로 사용합니다.

글로벌 매개변수

이 매개변수들은 모든 테마에서 인식됩니다. 각 테마는 자체 기본값을 제공합니다.

매개변수 별칭 타입 기본값 설명
theme t string default 테마 ID (테마 참조)
mode — string auto sign (정적) 또는 flow (스크롤)
color c hex 테마별 텍스트 색상 (색상 형식)
bg — hex 테마별 배경 색상
speed — number 테마별 스크롤 속도 (px/s, 플로우 모드)
direction dir string left 스크롤 방향: left 또는 right
font — string 테마별 폰트 패밀리 이름
scale — number 1 디스플레이 스케일 배율 (0.1–1). 스케일 참조
fill — hex 테마별 카드 면 배경 (카드 테마, scale < 1일 때)
wakelock w boolean true 화면 켜짐 유지 (절전 방지)
cursor cur number — 커서 자동 숨김 지연 시간 (밀리초)
lang l string auto UI 언어: en, zh, ja, ko, es, fr, de

색상 형식

색상은 # 접두사 없이 16진수 값으로 지정합니다:

  • 6자리 RRGGBB — 예: ff0000 (빨강)
  • 8자리 AARRGGBB — 예: 80ff0000 (50% 투명 빨강)
led.run/HELLO?color=00ff41          초록색 텍스트
led.run/HELLO?bg=1a0033&c=ff00ff   보라색 배경, 마젠타 텍스트

스케일 & 카드 테마

scale 매개변수(0.1–1)는 디스플레이 크기를 제어합니다. 동작은 테마 유형에 따라 다릅니다:

카드 테마

카드 테마(broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix)는 내부 래퍼에 CSS transform: scale()을 사용합니다. scale < 1일 때, 카드가 투명 배경 위에 중앙 정렬되어 떠 있습니다. fill 매개변수로 카드 면 색상을, bg로 페이지 배경을 설정합니다.

확장 테마

그 외 모든 테마는 scale을 폰트 크기 배율로 적용합니다: 자동 맞춤 결과에 스케일 값을 곱합니다. 배경 효과는 전체 화면을 유지합니다.

인터랙션

사이니지가 표시되면 키보드, 마우스 또는 터치로 상호작용할 수 있습니다.

키보드

  • Space 스크롤 일시정지 / 재개 (플로우 모드)
  • F 전체 화면 전환
  • Esc 전체 화면 종료

마우스 / 터치

  • 클릭 / 탭 한 번 일시정지 / 재개
  • 더블 클릭 / 탭 전체 화면 전환

툴바

화면 하단에 플로팅 툴바가 나타납니다 (커서와 함께 자동 숨김). 다음 기능을 제공합니다:

  • 전체 화면 — 전체 화면 모드 진입 또는 종료
  • 회전 — 디스플레이를 90° 회전 (가로 화면에서 세로 사인에 유용)
  • 캐스트 — Presentation API를 통해 외부 디스플레이로 전송
  • 공유 — 공유를 위해 현재 URL 복사

캐스팅
