led.run 문서

어떤 화면이든 텍스트 사인, 조명 효과, 사운드 비주얼라이저, 시계 디스플레이로 바꿔보세요. 앱도, 계정도 필요 없습니다 — URL 하나면 충분합니다.

작동 원리

led.run은 URL을 사용하여 디스플레이를 만듭니다. 경로와 쿼리 문자열로 모든 것을 설정합니다. 아무 브라우저에서 URL을 열면 디스플레이가 바로 준비됩니다 — 서버 처리도, 계정도 필요 없습니다.

텍스트 사인

led.run/HELLO?theme=neon&color=ff00ff
텍스트 (URL 경로) 설정 (쿼리 파라미터)

조명 효과

led.run/light?t=disco&speed=3
제품 접두어 효과 & 설정

사운드 비주얼라이저

led.run/sound?t=bars&sensitivity=5
제품 접두어 비주얼라이저 & 설정

시계 디스플레이

led.run/time?t=digital&format=24h
제품 프리픽스 시계 및 설정

URL 프로토콜

네 가지 제품, 네 가지 URL 패턴:

Text:   led.run/HELLO?t=neon              경로 = 텍스트 내용
        led.run/text/HELLO?t=neon         명시적 /text/ 접두어
Light:  led.run/light?t=disco             제품 접두어 + 파라미터
Sound:  led.run/sound?t=bars              제품 접두어 + 파라미터
Time:   led.run/time?t=digital            제품 접두어 + 파라미터
  • Text — / 뒤의 URL 경로가 표시 텍스트가 됩니다. 공백은 + 또는 %20으로 입력합니다.
  • Light, Sound & Time — 예약된 경로 접두어를 사용합니다. 모든 설정은 쿼리 파라미터로 합니다.
  • 모든 파라미터는 선택적인 “기본 설정 힌트”입니다 — 테마/효과/비주얼라이저가 사용 여부를 결정합니다.
예약된 접두어: text, light, sound, time은 예약어입니다. “light”라는 단어를 텍스트로 표시하려면 led.run/text/light를 사용하세요.

Text — 개요

Text 제품은 모든 URL 경로를 전체 화면 디스플레이로 변환합니다. 각각 특정 분위기와 사용 사례에 맞게 설계된 20개의 기본 테마가 포함되어 있습니다. 테마는 정적 사인과 스크롤 플로우의 두 가지 디스플레이 모드를 지원합니다.

led.run/HELLO                          간단한 사인
led.run/DO+NOT+DISTURB?t=do-not-disturb   테마 선택
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   커스텀 색상
led.run/WELCOME?mode=flow&speed=120      스크롤 자막

디스플레이 모드

텍스트 테마는 두 가지 디스플레이 모드를 지원합니다:

사인 모드

정적 전체 화면 디스플레이. 텍스트가 사용 가능한 공간에 맞게 자동 조절됩니다. “OPEN”, “ON AIR”, “DO NOT DISTURB” 같은 짧은 메시지에 적합합니다.

플로우 모드

스크롤 자막. 텍스트가 화면을 가로질러 연속으로 스크롤됩니다. 긴 메시지, 공지사항, 티커에 적합합니다.

자동 감지: mode 파라미터가 설정되지 않으면, 대부분의 테마는 짧은 텍스트(10자 이하)에는 사인 모드를, 긴 텍스트에는 플로우 모드를 기본으로 사용합니다.

텍스트 파라미터

이 파라미터들은 텍스트 테마에서 인식됩니다. 각 테마는 자체 기본값을 제공합니다.

파라미터 별칭 타입 기본값 설명
theme t string default 테마 ID (테마 참조)
mode — string auto sign (정적) 또는 flow (스크롤)
color c hex 테마별 텍스트 색상 (색상 형식)
bg — hex 테마별 배경 색상
speed — number 테마별 스크롤 속도 px/s (플로우 모드)
direction dir string left 스크롤 방향: left 또는 right
font — string 테마별 폰트 패밀리 이름
scale — number 1 디스플레이 스케일 배수 (Text: 0.1–1, Time: 0.1–3). 스케일 참조
fill — hex 테마별 카드 표면 배경 (카드 테마, scale < 1일 때)
wakelock w boolean true 화면 켜짐 유지 (절전 방지)
cursor cur number — 커서 자동 숨김 지연 시간 (밀리초)
lang l string auto UI 언어: en, zh, ja, ko, es, fr, de

라이트 — 개요

라이트 제품은 모든 화면을 전체 화면 조명 이펙트로 변환합니다. 손전등, 스트로브, 무드 조명 또는 앰비언트 디스플레이로 사용할 수 있습니다. 18가지 내장 이펙트는 실용적인 것(solid, SOS)부터 분위기 있는 것(aurora waves, lava lamp)까지 다양합니다.

led.run/light?t=solid              흰색 손전등
led.run/light?t=disco&speed=5      빠른 디스코 조명
led.run/light?t=candle&warmth=9    따뜻한 촛불
led.run/light?t=aurora-waves       오로라 조명 밴드
참고: /light에 ?t=가 없으면 랜딩 페이지가 열립니다. 공유 링크에는 항상 ?t=가 포함됩니다.

라이트 매개변수

라이트 이펙트에서 인식하는 공통 매개변수:

사운드 — 개요

사운드 제품은 마이크로 구동되는 실시간 오디오 시각화를 생성합니다. 12가지 내장 비주얼라이저가 소리를 매혹적인 비주얼로 변환합니다. 클래식 스펙트럼 바부터 시네마틱 보케 필드까지 다양합니다.

led.run/sound?t=bars              스펙트럼 분석기
led.run/sound?t=ocean&waveCount=10 더 많은 파도의 바다
led.run/sound?t=spikes            방사형 스파이크
마이크 권한: 사운드 비주얼라이저는 마이크 접근이 필요합니다. 처음 사용 시 브라우저에서 권한을 요청합니다. 오디오는 로컬에서 처리됩니다 — 녹음되거나 전송되지 않습니다.

시계 — 개요

시계 제품은 화면을 풀스크린 시계 디스플레이로 변환합니다. 클래식 7세그먼트 LED부터 닉시 튜브, 해시계까지 14가지 내장 시계 테마 중에서 선택할 수 있습니다. 각 테마는 고유한 비주얼 스타일로 현재 시간을 렌더링합니다.

led.run/time?t=digital                 기본 디지털 시계
led.run/time?t=nixie&color=ff8833      닉시 튜브 시계
led.run/time?t=analog&style=modern     모던 아날로그 시계
led.run/time?t=lcd&format=12h          LCD 시계, 12시간
참고: /time에 ?t=가 없으면 랜딩 페이지로 이동합니다. 공유 링크에는 항상 ?t=가 포함됩니다 (예: led.run/time?t=digital).

Color Format

Colors are specified as hex values without the # prefix:

  • 6-digit RRGGBB — e.g. ff0000 for red
  • 8-digit AARRGGBB — e.g. 80ff0000 for 50% transparent red
led.run/HELLO?color=00ff41          Green text
led.run/HELLO?bg=1a0033&c=ff00ff   Purple bg, magenta text

Scale & Card Themes

The scale parameter controls display size. Its behavior depends on the theme type:

Card Themes

Card themes (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) use CSS transform: scale() on an inner wrapper. When scale < 1, the card floats centered on a transparent background. Use the fill param to set the card face color, and bg for the page background.

Extended Themes

All other themes apply scale as a font-size multiplier: the auto-fit result is multiplied by the scale value. Background effects remain fullscreen.

Time Product

Clock themes support scale (0.1–3), position, and padding as layout parameters. When scale ≠ 1 or padding > 0, a wrapper is created inside the container with real dimensions (not CSS transform), so themes render at the correct size. Values < 1 shrink the clock; values > 1 enlarge it beyond the container, with overflow:hidden clipping the excess. Use position to control which portion of the enlarged clock is visible (e.g., top-left anchors the top-left corner). padding adds spacing from container edges.

led.run/time?t=digital&scale=0.5&position=top-left&padding=3
led.run/time?t=digital&scale=2&position=center            — 2× zoom, centered
led.run/time?t=digital&scale=2&position=top-left           — 2× zoom, top-left anchor

Casting

The Cast feature uses the W3C Presentation API to display your content on a secondary screen. Currently supported in Chrome and Edge.

  • Click the Cast button in the toolbar to select a display
  • The receiver shows a clean display without toolbar or controls
  • The connection persists across page navigations
The Cast button only appears in browsers that support the Presentation API.
