Documentation led.run

Transformez n'importe quel écran en panneau d'affichage numérique professionnel. Pas d'application, pas de compte — juste une URL.

Comment ça marche

led.run transforme les URL en panneaux d'affichage numériques. Le chemin est votre texte, la chaîne de requête est votre configuration. Ouvrez l'URL dans n'importe quel navigateur et votre panneau est prêt.

led.run/HELLO?theme=neon&color=ff00ff
Texte (chemin URL) Configuration (paramètres de requête)

Le fonctionnement est simple : l'URL est analysée en texte et paramètres, un moteur de rendu de thème prend le contrôle de l'écran, et votre panneau est affiché. Tout se passe côté client — aucun traitement serveur.

Protocole URL

https://led.run/[text]?[params]
  • Texte — le chemin URL après /. Les espaces peuvent être encodés en + ou %20.
  • Paramètres — chaîne de requête standard. Tous les paramètres sont des « indications de préférence » optionnelles — les thèmes décident de les utiliser ou non.

Exemples

led.run/HELLO                          Panneau simple
led.run/DO+NOT+DISTURB?t=do-not-disturb   Sélection du thème
led.run/OPEN?t=neon&c=ff2d78&bg=0a0008   Couleurs personnalisées
led.run/WELCOME?mode=flow&speed=120      Défilement

Modes d'affichage

Les thèmes prennent en charge deux modes d'affichage :

Mode panneau (Sign)

Affichage statique en plein écran. Le texte est automatiquement ajusté pour remplir l'espace disponible. Idéal pour les messages courts comme « OPEN », « ON AIR » ou « DO NOT DISTURB ».

Mode défilant (Flow)

Défilement continu. Le texte défile en continu à travers l'écran. Idéal pour les messages longs, les annonces ou les bandeaux d'information.

Détection automatique : Si aucun paramètre mode n'est défini, la plupart des thèmes utilisent par défaut le mode panneau pour les textes courts (≤10 caractères) et le mode défilant pour les textes plus longs.

Paramètres globaux

Ces paramètres sont reconnus par tous les thèmes. Chaque thème fournit ses propres valeurs par défaut.

Paramètre Alias Type Défaut Description
theme t string default Identifiant du thème (voir Thèmes)
mode — string auto sign (statique) ou flow (défilant)
color c hex par thème Couleur du texte (format des couleurs)
bg — hex par thème Couleur de fond
speed — number par thème Vitesse de défilement en px/s (mode défilant)
direction dir string left Sens de défilement : left ou right
font — string par thème Nom de la famille de police
scale — number 1 Multiplicateur d'échelle (0.1–1). Voir Échelle
fill — hex par thème Fond de la face de la carte (thèmes carte, quand scale < 1)
wakelock w boolean true Maintenir l'écran allumé (empêche la mise en veille)
cursor cur number — Délai de masquage automatique du curseur en millisecondes
lang l string auto Langue de l'interface : en, zh, ja, ko, es, fr, de

Format des couleurs

Les couleurs sont spécifiées en valeurs hexadécimales sans le préfixe # :

  • 6 chiffres RRGGBB — ex. ff0000 pour le rouge
  • 8 chiffres AARRGGBB — ex. 80ff0000 pour un rouge transparent à 50%
led.run/HELLO?color=00ff41          Texte vert
led.run/HELLO?bg=1a0033&c=ff00ff   Fond violet, texte magenta

Échelle & thèmes carte

Le paramètre scale (0.1–1) contrôle la taille d'affichage. Son comportement dépend du type de thème :

Thèmes carte

Les thèmes carte (broadcast, street-sign, wood, do-not-disturb, marquee, dot-matrix) utilisent transform: scale() CSS sur un conteneur interne. Quand scale < 1, la carte flotte centrée sur un fond transparent. Utilisez le paramètre fill pour définir la couleur de la face de la carte, et bg pour le fond de la page.

Thèmes étendus

Tous les autres thèmes appliquent scale comme multiplicateur de taille de police : le résultat de l'ajustement automatique est multiplié par la valeur d'échelle. Les effets de fond restent en plein écran.

Interactions

Une fois un panneau affiché, vous pouvez interagir avec lui au clavier, à la souris ou au toucher.

Clavier

  • Espace Mettre en pause / reprendre le défilement (mode défilant)
  • F Basculer en plein écran
  • Echap Quitter le plein écran

Souris / Tactile

  • Simple clic / appui Mettre en pause / reprendre
  • Double clic / appui Basculer en plein écran

Barre d'outils

Une barre d'outils flottante apparaît en bas de l'écran (se masque automatiquement avec le curseur). Elle propose :

  • Plein écran — entrer ou quitter le mode plein écran
  • Rotation — faire pivoter l'affichage de 90° (utile pour les panneaux portrait sur écrans paysage)
  • Diffusion — diffuser vers un écran externe via l'API Presentation
  • Partage — copier l'URL actuelle pour le partage

Diffusion

La fonction de diffusion utilise l'API W3C Presentation pour afficher votre panneau sur un écran secondaire. Actuellement prise en charge dans Chrome et Edge.

  • Cliquez sur le bouton Diffusion dans la barre d'outils pour sélectionner un écran
  • Le récepteur affiche un écran propre sans barre d'outils ni contrôles
  • La connexion persiste lors des navigations de page
Le bouton Diffusion n'apparaît que dans les navigateurs prenant en charge l'API Presentation.
